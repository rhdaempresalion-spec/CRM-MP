Sistema Online - PagamentosMP

Gerador de PIX

Cada requisição gera um PIX novo e único

Valor do PIX
R$ 12,90

Como funciona:

1. Preencha os dados do cliente

2. Clique em "Gerar PIX"

3. O sistema gera o PIX via API PagamentosMP e envia para o CRM

4. O código PIX copia e cola é enviado de volta ao CRM


Endpoints:

POST /webhook - Receber webhook do CRM

POST /api/gerar-pix - Gerar PIX via API

POST /webhook/mp-callback - Callback PagamentosMP
